## My personal website

### [sericaia.me](https://www.sericaia.me)

Uses [Gatsby](https://www.gatsbyjs.org/) with [React.js](http://reactjs.org/) and [GraphQL](https://graphql.org/).

### How to run

> `$ yarn`

> `$ gatsby develop`
